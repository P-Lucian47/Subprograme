Introducere în Subprograme

Învață cum funcționează funcțiile și subprogramele în C++.

Ce este un subprogram?

Un subprogram reprezintă o secvență de instrucțiuni care realizează o anumită sarcină.

În C++, subprogramele sunt numite funcții.

Exemplu simplu


#include <iostream>
using namespace std;

void salut()
{
    cout << "Salut!";
}

int main()
{
    salut();

    return 0;
}

Cum funcționează?

  • void → funcția nu returnează nimic
  • salut() → numele funcției
  • cout → afișează textul
  • Funcția este apelată în main()
