Introducere în Subprograme

Învață cum funcționează funcțiile și subprogramele în C++.

Ce este un subprogram?

Un subprogram reprezintă o secvență de instrucțiuni care realizează o anumită operație.

Avantajele utilizării subprogramelor:

  • cod mai organizat;
  • evitarea repetiției;
  • program mai ușor de înțeles;
  • depanare mai simplă.

Important

Un subprogram trebuie apelat pentru a fi executat.

Exemplu de funcție


#include <iostream>
using namespace std;

void mesaj()
{
    cout << "Invatam subprograme!";
}

int main()
{
    mesaj();

    return 0;
}

Explicația codului

  • void → funcția nu returnează valori.
  • mesaj() → numele funcției.
  • main() → funcția principală.
  • mesaj(); → apelul funcției.

Funcție cu return


#include <iostream>
using namespace std;

int suma(int a, int b)
{
    return a + b;
}

int main()
{
    cout << suma(4, 5);

    return 0;
}
