Funcții cu Parametri

Subprogramele devin utile atunci când rezolvăm probleme complexe și repetitive.

Funcții cu parametri

Parametrii permit transmiterea valorilor către o funcție. 


#include <iostream>
using namespace std;

void afiseaza(int x)
{
    cout << x;
}

int main()
{
    afiseaza(10);

    return 0;
}

Explicație

  • int x este parametrul funcției.
  • Valoarea 10 este transmisă funcției.
  • Funcția afișează valoarea primită.

Funcții care returnează valori

Unele funcții efectuează calcule și întorc rezultatul. 


#include <iostream>
using namespace std;

int patrat(int x)
{
    return x * x;
}

int main()
{
    cout << patrat(6);

    return 0;
}

Exemplu de problemă mai complexă

Calcularea sumei cifrelor unui număr folosind subprograme. 


#include <iostream>
using namespace std;

int sumaCifre(int n)
{
    int s = 0;

    while(n > 0)
    {
        s = s + n % 10;
        n = n / 10;
    }

    return s;
}

int main()
{
    cout << sumaCifre(5724);

    return 0;
}

Observație

Subprogramele sunt foarte utile în problemele care necesită multe calcule repetitive.
